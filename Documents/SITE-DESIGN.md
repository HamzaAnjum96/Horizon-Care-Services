# Horizon Care Services — Site Design Brief (Current Scope)

## Status

This document has been updated to match the current implementation strategy.

The active phase is now a **homepage look-and-feel refinement only**.

For the execution roadmap, see:
- [`HOMEPAGE-SPRINT-PLAN.md`](./HOMEPAGE-SPRINT-PLAN.md)
- [`CLIENT-DEV-DISCUSSION.md`](./CLIENT-DEV-DISCUSSION.md)

---

## Current Scope

- Improve the visual quality and personality of the existing homepage.
- Keep trust-first clarity for referral and family audiences.
- Use restrained, purposeful animation.
- Keep a short footer on the homepage.

## Out of Scope (for this phase)

- Creating new pages.
- Expanding sitemap or navigation depth.
- Rolling out long utility footer patterns.

## Notes

Previous versions of this document contained broader multi-page expansion concepts.
Those are intentionally deferred until the homepage phase is complete.
